# 시장 분석 및 차별화 전략 (Market Analysis)

## 1. 현재 시장 현황
현재 국내 테이블 오더 시장은 크게 **태블릿 기반**과 **QR/모바일 기반**으로 나뉩니다.

| 구분 | 태블릿 기반 (예: 티오더) | QR/모바일 기반 (예: 네이버/토스) | 제안 모델 (고정비 기반 QR 오더) |
| :--- | :--- | :--- | :--- |
| **하드웨어** | 테이블당 태블릿 설치 필수 | 하드웨어 필요 없음 (QR) | **하드웨어 필요 없음 (QR)** |
| **유지보수** | 배선 공사, 배터리 충전, 고장 관리 등 발생 | 매우 낮음 | **매우 낮음** |
| **비용 구조** | 기기 구매/리스료 + 월 이용료 | 네이버/토스 결제 수수료 중심 | **고정 월정액 (예: 3만원)** |
| **강점** | 압도적인 시인성, 직관적인 UI | 초기 비용 제로, 플랫폼 연동 | **가장 낮은 총소유비용(TCO), POS 독립** |

## 2. 사용자 아이디어의 실현 가능성 (Feasibility)
사용자께서 말씀하신 아이디어는 **기술적으로 충분히 가능하며, 시장의 확실한 가려운 곳(Pain Point)을 긁어주는 모델**입니다.

### 왜 가능한가?
1.  **전력 및 유지보수 문제 해결**: 태블릿 오더의 가장 큰 단점은 테이블마다 전기를 끌어오거나 매일 밤 배터리를 충전해야 한다는 점입니다. 사용자 모델은 100% 고객 핸드폰을 쓰므로 가게의 전기/관리 부담이 0입니다.
2.  **고정비 모델의 매력**: 매출이 늘어날수록 수수료가 커지는 PG/플랫폼 수수료 기반보다, 한 달에 딱 정해진 금액(3만원 등)만 내는 구독형 모델은 소상공인에게 심리적/경제적 안정감을 줍니다.
3.  **플랫폼 종속 탈피**: 네이버나 토스 오더는 특정 결제 수단이나 POS 시스템을 강제하는 경우가 많습니다. '독립적인 웹 기반 QR 오더'는 어떤 POS를 쓰든 상관없이 주문만 전달해주면 되므로 범용성이 높습니다.

## 3. 수익 모델의 혁신: "Stop Double Fees (Bundle vs Segmented)"
상점 입장에서 본 시스템이 매력적인 결정적 이유는 '이중 부담'을 없앴기 때문입니다.

- **기존 분절형 모델**: POS 업체 수수료 + 오더 시스템 업체 수수료 + 결제사 수수료 = **수익성 악화 및 관리 복잡**.
- **우리의 통합 모델**: 
    - **POS + 모바일 오더를 하나의 팩**으로 제공.
    - 결제 수수료는 통합 시스템 하에서 최적화하고, 모바일 오더 기능은 **합리적인 고정비(월 3만원 등)**로 제공.
- **시장의 파급력**: "이미 내고 있는 수수료 내에서 모든 디지털 주문 기능을 누릴 수 있다"는 점이 강력한 시장 전파력을 가집니다.
